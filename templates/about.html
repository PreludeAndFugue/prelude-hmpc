Comments

You can add some mark-up to comments using the Markdown format.

To add links to your comments, you can either surround the link with angle brackets. This input 

<http://www.bbc.co.uk>
will produce this 
http://www.bbc.co.uk

Or you can turn text into a link. This input 

[The BBC](http://www.bbc.co.uk)
will produce this 
The BBC

To emphasise words, you can surround words with asterisks, like so:

*single asterisks*
**double asterisks**

The submitted comment will then look like the following 

single asterisks
double asterisks

There is much more to the markdown format, but this should be sufficient for our needs.
